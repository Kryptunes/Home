## How to use

1. Copy `posts/template/` → `posts/YYYY-MM-your-topic/`
2. Drop PNGs (**1080×1350**, LinkedIn carousel) into `slides/`
3. Name slides `01.png`, `02.png`, … in order
4. Fill `notes.md` — hook, bullets, caption draft
5. Upload `slides/` to LinkedIn; paste caption from `notes.md`

---

## Structure

```
posts/              one folder per carousel
  template/         copy this to start a new post
assets/brand/       logo, colours (optional)
```
